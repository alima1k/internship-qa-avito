## 1. Создание объявления POST {{baseUrl}}/api/1/item

## Предусловия

 - **Эндпоинт:** `{{baseUrl}}/api/1/item`
 - **Метод:** `POST`
 - **Заголовки:** `Content-Type: application/json`

**Схема 1:** Схема тела POST запроса на {baseUrl}/api/1/item.

```json 
{
  "sellerID": "<integer>",
  "name": "<string>",
  "price": "<integer>",
  "statistics": {
    "likes": "<integer>",
    "viewCount": "<integer>",
    "contacts": "<integer>"
  }
}
```
**Схема 2:** Схема успешного ответа 200 OK для POST {{baseUrl}}/api/1/item.

```json 
{
  "id": "<string>",
  "sellerId": "<integer>",
  "name": "<string>",
  "price": "<integer>",
  "statistics": {
    "likes": "<integer>",
    "viewCount": "<integer>",
    "contacts": "<integer>"
  },
  "createdAt": "<string>"
}
```
**Схема 3:** Схема ответа с ошибкой. Предполагается, что ключи в `messages` соответствуют именам полей с ошибками.

```json 
{
  "result": {
    "messages": {
      "culpa_b92": "<string>",
      "enim_24f": "<string>",
      "mollit_aa": "<string>"
    },
    "message": "<string>"
  },
  "status": "<string>"
}
```
