# Задание 1: Скриншот с багами

## P0 - Критический приоритет

### P0 - Некорректная работа фильтров поиска (пол, цена, порода, город)
Фильтры поиска работают некорректно. При введенных значениях "мальчик", "1000 - 30000", "Персидская порода" и "Москва",
в результате поиска пользователь может наблюдать объявления с девочками, с ценой, значительно ниже указанной (500),
из других городов (Новосибирск) и другой породы ("Экзотическая короткошерстная").

**Приоритет: P0.** Фильтрация - одна из самых основных функциональностей страницы поиска. Некорректная работа фильтров
делает невозможным получение ожидаемых результатов и полностью нарушает основной пользовательский сценарий.

### P0 - Отображается объявление, не относящееся к категории поиска
При выбранной категории кошек, на странице присутствует игрушка.

**Приоритет: P0.** Это одна из основных функциональностей страницы поиска. Пользователь не может по-другому 
задать категорию

### P0 - Некорректное отображение города в чек-боксе сортировки
Пользователь выбрал город по умолчанию Москва. При выдаче результатов вместо чекбокса "Сначала из Москвы" отображается "Сначала из Можайска".

**Приоритет: P0.** Пользователь не имеет возможности отсортировать объявления по Москве, если чекбокс запускает сортировку по Можайску.
Это нарушает сценарий поиска.

## P1 - Высокий приоритет

### P1 - На странице отображается сообщение об ошибке при корректной загрузке контента
На странице отображается сообщение об ошибке "Ошибка, обновите страницу", хотя на первый
взгляд вся функциональность страницы работает.

**Приоритет: P1.** Сообщение об ошибке отображается без явной причины и создает впечатление сбоя системы.
Пользователь может решить, что система работает нестабильно, и покинуть страницу, даже если фактически всё функционирует. 

### P1 - В заголовке страницы показывается количество выдаваемых объявлений, отличное от фактического
В заголовке указано 19 объявлений, фактически их 8. После ввода фильтров также отображается, что в системе 8 
подходящих объявлений, что подтверждает несоответствие значения в заголовке.

**Приоритет: P1.** Некорректное количество формирует у пользователя негативный опыт. Ему может показаться, что какие-то объявления 
скрыты или не загрузились. 

## P2 - Средний приоритет

### P2 - В хлебных крошках страницы поиска показывается неверная иерархия категорий
В хлебных крошках сначала отображается порода (Персидская), затем вид животного (Кошки). Это создает впечатление, 
что "Кошки" являются подкатегорией "Персидской породы".

**Приоритет: P2.** Ошибка влияет на восприятие структуры каталога. Пользователя это может ввести в заблуждение, 
однако, если ссылки работают корректно, сценарий поиска воспроизводится.

### P2 - В выборе возраста кошек отображаются некорректные опции
Вместо опций, подходящих кошкам (взрослая, котенок), система предлагает выбрать из возрастов собак (взрослая, щенок).

**Приоритет: P2.** Поиск осуществляется корректно, но пользователь может не связать опцию "щенок" с кошками и в итоге не
будет уверен в результатах выдачи объявлений.

### P2 - В разделе "Подобрали для Вас" предлагаются объявления, несоответствующие поисковому запросу
При поиске персидской породы, в разделе "Подобрали для Вас" предлагаются коты бенгальской породы. Эти две породы не являются
родственными, рекомендации не учитывают поисковый запрос.

**Приоритет: P2.** Баг не влияет на поиск, однако снижает качество рекомендаций. Это сказывается на вовлеченности пользователей.

### P2 - При наведении курсора на кнопку добавления товара в избранное, показывается некорректная всплывающая подсказка
При наведении курсора на кнопку (иконка сердца) вместо «Добавить в избранное», пользователь видит текст «Не показывайте мне похожие объявления».

**Приоритет: P2.** Подсказка противоречит ожиданиям пользователя от системы и может вызвать сомнение. Но благодаря иконке 
действие все еще остается интуитивно понятным.

### P2 - Дата публикации не отображается у всех объявлений
При выдаче результатов поиска, дата публикации присутствует только у двух объявлений из восьми.

**Приоритет: P2.** Дата публикации играет большую роль при решении просмотреть объявление. Ее отсутствие снижает информативность
выдачи, однако никак не мешает работоспособности поиска.

### P2 - Орфографическая ошибка в хедере страницы
В разделе "Карьера в Авито" допущена орфографическая ошибка: "Коръера в Авито".

**Приоритет: P2.** Не затрагивает основную функциональность страницы поиска, однако ошибки находятся в хедере, общем
для всех страниц сайта. Влияет на восприятие бренда пользователями.

## P3 - Низкий приоритет

### P3 - Пропущена буква в подписи к кнопке на странице поиска
В подписи к кнопке вместо "Уведомлять о новых" отображается "Уведомять о новых".

**Приоритет: P3.** Не затрагивает основную функциональность страницы поиска. Косметический недочет, смысл фразы остается понятным.

### P3 - Некорректное выравнивание карточек при выдаче
Карточки отображаются с нарушением выравнивания. Второй ряд смещен относительно первого. Отступы карточек неравномерные.

**Приоритет: P3.** Не затрагивает основную функциональность страницы поиска. Влияет только на визуальное восприятие.

### P3 - Окно с непрочитанным сообщением показывается не полностью
В правом нижнем углу уведомление о непрочитанном сообщении обрезается границами экрана.

**Приоритет: P3.** Текст остается читаемым, частично ухудшает восприятие интерфейса.
